Site web d'information sur la ville d'Édimbourg pour assister les visiteurs francophones dans le déroulement de leur voyage.

Besoin du projet.

Le projet.

Ce projet est né d'un constat : l'absence d'informations en français sur la ville d'Édimbourg. En effet, les sites web existants ne proposent pas de traduction pour les visiteurs francophones. De plus, les sites en anglais présentent souvent un design peu moderne et ne proposent pas une expérience utilisateur optimale. C'est pourquoi la création d'un site web d'information sur Édimbourg, destiné spécialement aux francophones, s'avère indispensable.

La création d'un site web d'information sur Édimbourg pour les visiteurs francophones est un projet ambitieux qui vise à améliorer la qualité de l'information disponible pour ce public. Le site sera conçu de manière à offrir une expérience utilisateur optimale, avec une interface moderne et intuitive. Les informations sur les sites touristiques, les événements culturels, les activités pour toute la famille, les restaurants, les bars, les hôtels, les transports publics et les horaires des attractions touristiques seront présentées de manière claire et précise. Ce site web sera donc un véritable guide pour les visiteurs francophones souhaitant explorer Édimbourg.

01

La recherche.

Mon client a fourni un gros travail de récolte d'information auprès de toutes les sources fiables disponibles. Il était donc ensuite question d'arranger ces informations de manière optimale pour faciliter la navigation du visiteur et réduire au maximum son temps d'accès à ces informations.

02

Choix du design.

L'objectif principal de ce site web est de fournir des informations simples, rapides et facilement accessibles aux visiteurs francophones souhaitant découvrir Édimbourg. Pour atteindre cet objectif, une approche minimaliste et moderne a été choisie, éliminant tout contenu superflu. Ainsi, les visiteurs peuvent facilement trouver les informations dont ils ont besoin pour planifier leur voyage sans avoir à chercher longtemps.

Le mot de la fin.

C'éait donc un véritable plaisir de rendre accessible toutes ces informations de manière plus moderne et davantage attrayante visuellement ! Merci Sonia :)
